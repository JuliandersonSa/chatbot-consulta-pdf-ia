Chatbot Terminal com Consulta de PDFs

Um assistente de terminal inteligente que lê, resume e responde perguntas sobre seus documentos PDF.

Sobre o Projeto

Este projeto foi desenvolvido para facilitar o estudo, leitura e consulta de arquivos PDF diretamente via terminal. Utilizando a API da OpenAI, o chatbot gera resumos dos documentos e permite ao usuário interagir com eles em linguagem natural.

Funcionalidades

  • Leitura e resumo automático de arquivos PDF
  • Armazenamento e carregamento de sessões de conversa
  • Gerenciamento de resumos e conversas exportadas
  • Interação em linguagem natural com base no conteúdo resumido
  • Exportação do histórico de conversa em PDF

Como Usar

  1. Clone o repositório: git clone https://github.com/JuliandersonSa/chatbot-consulta-pdf-ia.git
  2. Instale os requisitos com pip install -r requirements.txt
  3. Configure sua chave da OpenAI no arquivo .env
  4. Execute com python3 main.py

Comandos Disponíveis

O chatbot aceita comandos iniciados por /. Abaixo estão os principais:

  • /lerpdf <arquivo.pdf>: lê o PDF especificado, gera um resumo e ativa como contexto da sessão atual.
  • /nova_sessao <nome>: cria e ativa uma nova sessão.
  • /carregar_sessao <nome>: carrega uma sessão previamente salva.
  • /listar_sessoes: exibe todas as sessões disponíveis.
  • /excluir_sessao <nome>: remove a sessão informada (exceto a ativa).
  • /listar_resumos: exibe todos os resumos de PDF salvos.
  • /carregar_resumo <ID ou número>: ativa um resumo salvo para uso em conversas.
  • /excluir_resumo <ID ou número>: exclui um resumo específico.
  • /exportar_chat <nome>: gera um PDF com todo o histórico da sessão atual.
  • /listar_exports: mostra todos os PDFs exportados.
  • /carregar_export <nome>: mostra o caminho de um arquivo PDF exportado.
  • /excluir_export <nome>: remove um PDF exportado.
  • /limpar: limpa o histórico de conversa e contexto atual.
  • /ajuda: lista todos os comandos disponíveis.
  • /sair: salva e encerra o chatbot (opcionalmente exportando o histórico).

Como Gerar Histórico de Conversa

Durante uma sessão ativa, todas as interações com o chatbot são armazenadas. Para exportar esse histórico:

  1. Digite /exportar_chat nome_do_arquivo.
  2. O arquivo será salvo como PDF na pasta data/profiles/default/exports/NOME_SESSAO/.
  3. Você pode visualizar o caminho completo com /carregar_export nome_do_arquivo.

Essa funcionalidade é útil para registrar interações importantes e gerar relatórios de estudo automatizados.
